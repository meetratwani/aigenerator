## Common Issues

### "I don't see the startup message"
- The server is still training (first run = 10-15 minutes)
- Keep waiting and watch the terminal

### "Error or connection refused"
- Make sure you're in the right folder
- Try: `cd C:\Users\Meet\OneDrive\Desktop\ArpitSir`
- Then: `python server.py`

### "ModuleNotFoundError: No module named 'flask'"
- Install dependencies: `pip install -r requirements.txt`
- Then try again: `python server.py`

### "Port already in use"
- Another app is using port 5000
- Close other Python windows or change port
- Try: `python server.py` again
